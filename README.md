Hangman demo app (coding test)
==========

Simple java-based app for coding test


Under the hood
------

* Spring Boot 1.4, 
* JPA, H2
* Gradle build system
* AngularJS 1.5.8


How to run it?
------------

```
./gradlew bootRun
```

Navigate your browser to [http://localhost:8080](http://localhost:8080)
